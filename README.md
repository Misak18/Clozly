# Clozly — Онлайн-примерочная нового поколения (Frontend)

**Clozly** — это новая эра в сфере онлайн-шопинга. Главная проблема интернет-магазинов одежды — покупатели не знают, как вещь сядет именно на них. Наше решение полностью закрывает этот барьер, позволяя примерить одежду виртуально за считанные секунды.

В данном репозитории представлен **новый, улучшенный дизайн** нашей онлайн-примерочной и личного кабинета.


## Демонстрация
Посмотреть проект вживую и протестировать новый интерфейс можно здесь: **[clozly.ru](https://clozly.ru)**


## Ключевые преимущества Clozly

* ⚡ **Высокая скорость работы** — процесс генерации и примерки в среднем занимает всего **15 секунд**. Больше никаких долгих ожиданий.

* 🎨 **Продуманный UX/UI** — современный, интуитивно понятный и эстетичный дизайн, который встречает пользователя с первых секунд и затягивает в процесс.

* 📊 **Глубокая аналитика для бизнеса** — встроенная система трекинга действий клиентов. Владельцы магазинов могут в реальном времени отслеживать статистику примерок и видеть результаты вовлеченности.

* 📈 **Рост конверсии продаж** — благодаря интерактивной примерке пользователи принимают решение о покупке гораздо увереннее, что напрямую увеличивает процент продаж магазина и снижает количество возвратов.
